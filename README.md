# Pytorch Implementation of [kGCN](https://github.com/clinfo/kGCN)


## Why Pytorch version?

* More user-friendly
* More test
* More developer-friendly
* More new features of latest python.

## Requirements
* Python3 (>= 3.6)
* Pytorch (>= 1.0)
* Rdkit (>= 2020)
